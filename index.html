Space & Location Dashboard

Welcome to the Space & Location Dashboard! This website combines the power of the NASA API and the IP Geolocation API to provide you with an exciting view of space through the Astronomy Picture of the Day and information about your current location. Explore the wonders of the universe and learn more about your place on Earth!

How It Works

Connecting to NASA API

We connect to the NASA API to fetch the Astronomy Picture of the Day (APOD). This API provides daily images and information related to space and astronomy.

JavaScript code used:

fetch('https://api.nasa.gov/planetary/apod?api_key=DEMO_KEY')

NASA API Documentation

Connecting to IP Geolocation API

We connect to the IP Geolocation API to fetch the user's location based on their IP address. This API provides geographical information such as city and country based on the user's IP.

JavaScript code used:

fetch(`https://api.ipgeolocation.io/ipgeo?apiKey=${ipGeoApiKey}`)
